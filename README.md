In order to get this up and runnin' locally, please make sure that you have the following installed:

*Python 3.0+

Make a new database:

Open python3 in the web-app directory, and run create_db.py

Run the webserver:

* python3 runserver.py

Required modules:
* flask 
* sqlalchemy 
* flask-sqlalchemy
* flask-jsonpify 
* flask-restful

Enjoy! :)
